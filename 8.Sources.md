# State Space, Minecraft item list

Look at minecraftbible.com for information.

## Item Totals

**TOTAL ITEMS:** 1,489:
* 1,238 (64 stack)
* 26 (16 stack)
* 225 (1 stack)

---





## Notes

* `air`, `cave_air`, and `void_air` are not inventory items.
* Cake with candles (17 types) and partially eaten cakes are not included (these are block IDs, not item IDs).
* Compass and Recovery Compass NBT data are not included: world-related.
* loadestone compass has NBT Item ID: only base loadstone compass is included.
* A cast Fishing Rod is not included separately. The bobber determines whether it is cast, not the item itself (player/world state).
* A loaded Bow or Trident is constant player-input dependent and not a separate item ID.
