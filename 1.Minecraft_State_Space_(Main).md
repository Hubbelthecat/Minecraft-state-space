# Minecraft State Space

This is a research project dedicated to calculating the state space of Minecraft using the best available knowledge of game mechanics, data structures, and mathematics.

The goal is to construct a consistent, formal model of Minecraft as a system of distinguishable states.

If you notice any errors, missing details, or improvements in this repository or in related YouTube content, please leave feedback in the comment section of the relevant video (or the most recent state-space related upload).
