# Version State Space

This section contains links to version-specific state space calculations and comparisons.

Unlike the other major categories, the Version category does not represent a single Minecraft system. Instead, it serves as a framework for comparing and aggregating state space estimates across different Minecraft versions and editions.

This includes, but is not limited to:

* Modern Java Edition versions
* Modern Bedrock Edition versions
* Legacy Java Edition versions
* Legacy Bedrock Edition versions

As additional versions are analyzed, they will be added here.

## Current Status

At the time of writing, the Version category has not yet been formally modeled.

Future calculations are expected to become a major part of the project, as each version may contain different mechanics, items, blocks, entities, limits, and state-space assumptions.

Rather than introducing entirely new systems, this category combines and compares calculations from the other major categories within the constraints of each Minecraft version.

As additional versions are researched and formalized, the values above will be updated.
