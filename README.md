# Minecraft State Space Project

Hello, I am **Hubbelthecat**, and welcome to my grand project:


## Calculating the State Space of Minecraft

This project explores the full combinatorial state space of Minecraft systems — from inventory mechanics to NBT data, item interactions, and beyond.

---


## What this project is

At its core, this is an attempt to answer a deceptively simple question:

> How large is the state space of Minecraft?

To answer that, I am breaking the game down into formal mathematical systems, including:

* Inventory states
* Item and block representations
* NBT data structures
* Recursive container systems
* Game mechanics that influence distinguishable states

Along the way, I will probably:

* Suffer through combinatorics
* Stress my PC beyond reasonable limits
* learn quantum physics just to understand dirt and still not knowing what dirt is doing most of the time
